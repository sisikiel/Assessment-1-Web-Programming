CSS GUIDEBOOK

        
            

                

                    PROPERTY
                    DESCRIPTION
                

    
            

                background-color
                Specifies the background color of an element
            

    
            

                background-image
                Specifies one or more background images for an element
            


            

                background-repeat
                Sets if/how a background image will be repeated
            


            

                border
                A shorthand property for border-width, border-style, and border-block-color
            


            
            

                color
                Sets the color of text
            


            

                display
                Specifies how a certain HTML element should be displayed
            

            
            

                font-family
                Specifies the font family for text
            

    
            

                font-size
                Specifies the size of the text
            

    
            

                justify-content
                Specifies the alignment between the items inside a 
 flexible container when the items do not use all available space		
            

    
            

                list-style
                Sets all properties for a list in one declaration
            

    
            

                margin
                Sets all margin properties in one declaration
            

    
            

                padding
                A shorthand property for all the padding properties
            

    
            

                scroll-behavior
                Specifies whether to smoothly animate the scroll position in a scrollable box, instead of a straight jump
            


            

                text-align
                Specifies the horizontal alignment of the text
